The right panel is executable documentation of the native contract: handshake ack, correlated *_response replies, and __nbridge_batch__ unpacking.

Implemented contract

  • AndroidBridge.postMessage(json) — receives every web→native message as a JSON string
  • __nbridge_handshake__ → replies __nbridge_handshake_ack__
  • __nbridge_batch__ → unpacks payload.messages, handles each entry
  • device:getInfo → replies device:getInfo_response with the same id
  • toast:show → renders a fake Android toast
  • Replies delivered via window.sendBridgeMessage(json) after a randomized 80–350 ms delay
